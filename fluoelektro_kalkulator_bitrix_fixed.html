Sledeći korak

Pošaljite upit i naš tim za projektovanje će pripremiti detaljniju analizu i preporuku rešenja prilagođenu vašim potrebama.

// Glavne formule za proračun rasvete
površina = dužina × širina
adjustedUF = UF × (height > 5 ? 0.9 : 1.0) × (height > 8 ? 0.8 : 1.0)
potrebni_lumeni = površina × lux / (adjustedUF × MF)
broj_svetiljki = ceil(potrebni_lumeni / lumeni_po_svetiljki)
ukupna_snaga = broj_svetiljki × snaga_svetiljke
specifična_snaga = ukupna_snaga / površina (W/m²)
dosegnuta_osvetljenost = (broj_svetiljki × lumeni × adjustedUF × MF) / površina
// GRID LOGIC — Engineering-Grade Uniformity
cols = round(sqrt(count × (length/width)))
rows = ceil(count / cols)
totalPositions = cols × rows ≥ count
uniform_spacing_x = 100% / cols
uniform_spacing_y = 100% / rows
// Promenljive
UF = Faktor iskorišćenja (0.45 – 0.65)
MF = Faktor održavanja (0.65 – 0.80)
height = Visina montaže (m) — primeni korekciju ako > 5m
// Pametna pravila i preporuke
ako visina < 3m → paneli / downlight svetiljke
ako 3–5m → linearna suspenziona rasveta
ako > 5m → industrijska high-bay rasveta (primeni UF × 0.9)
ako > 8m → dodatna korekcija (primeni UF × 0.8)
ako kancelarija → UGR < 19 obavezno
ako industrija → optimizuj lm/W (min. 130 lm/W)
ako MF ≤ 0.70 → preporuči IP65 zaštitu
ako lm/W < 120 → Preporuka efikasnijeg modela

Ovaj proračun je informativnog karaktera. Za precizno projektovanje i usklađenost sa standardima neophodan je detaljan svetlotehnički proračun.
